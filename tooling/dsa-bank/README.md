# DSA Bank (VS Code extension)

Companion to the DSA_QUESTIONS launchpad. Handles `vscode://prateek.dsa-bank/open?...`
links: scaffolds `DSA/Solutions/<topic>/<id>-<slug>/` with a `README.md` (LeetCode
description) + starter `solution.cpp`, and opens them side by side.

Commands:
- **DSA: Submit to LeetCode & Push** — runs `tooling/lc-submit.sh` on the active file.
- **DSA: Open Launchpad** — opens `launchpad.html`.

Open the `DSA_QUESTIONS` repo folder in VS Code before clicking launchpad links.
